# Age Calculator

**Description:**  
A simple JavaScript-based Age Calculator that takes a user's date of birth as input and calculates their exact age in years, months, and days. Built with HTML, CSS, and JavaScript, providing a clean and responsive interface.

**Skills / Technologies Used:**  
- HTML  
- CSS  
- JavaScript (Date objects, functions, event listeners)  

**How to Run:**  
1. Clone or download the repository.  
2. Open `index.html` in a web browser.  
3. Enter your date of birth and click "Calculate" to see your age.
